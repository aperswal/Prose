# Prose

**[Download on the App Store](https://apps.apple.com/us/app/prose-hemingway-style-editor/id6771195769)**

A minimalist iOS writing app. Type a note, watch your readability grade update live in the toolbar, and toggle the eye icon to surface long sentences, passive voice, adverbs, inflated vocabulary, and filler words right where you wrote them.

Built with SwiftUI and SwiftData.

## Run

Open `WritingSimply.xcodeproj` in Xcode and run on an iPhone simulator.
